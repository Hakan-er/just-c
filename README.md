# just-c
C to C converter. It converts a C code to a real C code.

3 ways to run:
> ./just-c old.c new.c
>
> ./just-c old.c (new file will be newFile.c)
>
> ./just-c (new file will be newFile.c)

then you can compile and run the newFile.
